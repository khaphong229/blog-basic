# Language & Communication Rules

## Default Chat Language
- Assistant responses in chat must use Vietnamese with proper diacritics.
- Keep responses concise, clear, and actionable unless the user asks for more detail.

## Scope
- This rule applies to assistant chat replies only.
- Do not force Vietnamese for code, identifiers, command names, file paths, API payloads, or existing project content unless explicitly requested.

## Override Conditions
- If the user explicitly requests another language, follow the user's requested language.
- If quoting logs/errors/commands, keep original content unchanged and explain in Vietnamese when needed.
